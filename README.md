# 生鲜超市购物系统

网络应用开发课程实验作业，作者李隆耿，指导老师布社辉。

1、开发配置：

开发工具：IntelliJ IDEA 2019.2.4 x64

开发语言：java 1.8

操作系统：windows 10

数据库：MySQL 8.0.18

应用构建框架：Spring Boot

2、项目简介：

具有卖家和买家的水果生鲜超市购物系统。

3、实现功能：

顾客：

·用户注册、登录、注销；

·用户购买流程（浏览/查询->添加至购物车->删除/付款->查看已付款订单）

销售：

·商品目录的管理（添加、删除、修改）；

·销售报表统计和查询；

·客户的购买记录。
